GenericMiddlewareHandler

generic middleware handler

Constructor

new GenericMiddlewareHandler()

Generic middleware handler. it manages an array of functions and applies each of them on a target. Note that one instance of this handler can work on multiple object at the same time Sice the .apply method is indipendent of the state.
