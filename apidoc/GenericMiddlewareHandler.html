GenericMiddlewareHandler

GenericMiddlewareHandler

This class is used in various modules to handle middlewares.

A middleware is basically an array of functions

[fn1, fn2, fn3]

as we apply a set of parameters to a middleware (aka Middleware stack) these parameters will be passed to each function in this array. Functions have the ability to stop the execution of the next function.

params = ['foo', 1]
fn1(params)
fn2(params)
...

apply(params, index)

Source:
  • src/Middleware/generic.middleware.handler.js, line 94

apply a specific function from middleware array over a set of arguments

Note that .apply will continue to call functions on parameters until the end of the middleware array

should be called with .apply([...], 0)

Note that each middleware funciton has access to next and end callback. The names suggest what they do. next will immediately invoke the next function and end will end the execution of the current stack
